    一、岗位职责
    系统开发与维护
    负责产品后台系统开发、架构设计及核心模块编码，涉及高并发处理、性能优化等
    维护现有系统，处理问题排查、版本迭代及文档编写（如技术文档、用户手册）
    支持量化交易框架开发（如数字货币多交易所实盘系统）或安全检测类产品引擎开发
    技术实现与协作
    配合产品经理完成需求分析、UI交互实现及跨部门沟通
    参与分布式系统设计、Hadoop/Spark生态搭建或存储设备上层平台开发
    扩展性任务
    技术预研与前沿探索（如WebRTC优化、AI算法集成）
    担任职业培训师，传授Python开发技能（需具备3年以上经验及教学能力）
    二、任职要求
    技术能力
    编程基础
    精通Python语言及常用框架（Django、Flask、Tornado），熟悉多线程、异步编程
    掌握至少一种数据库（MySQL、Redis、MongoDB）及SQL优化技巧
    系统与工具
    熟练使用Linux系统及Shell脚本，熟悉Git、Docker等工具链
    具备Hadoop/Spark生态开发经验者优先（需熟悉HDFS、Hive等组件）
    行业专项技能
    量化开发：需熟悉numpy/pandas、机器学习库（如Scikit-learn）
    安全检测：了解网络安全协议及漏洞分析技术
    软技能
    良好的沟通能力、团队协作意识及抗压能力
    学习能力强，能快速掌握新技术（如多模态开发、微服务架构）
    三、教育背景
    学历要求
    多数岗位要求本科及以上学历（计算机、软件工程、电子工程等相关专业），部分初级岗位接受大专学历
    量化开发、存储系统等高级职位倾向硕士学历或具备数学/金融复合背景
    证书与培训
    计算机资格证书（如软考）或行业认证（如AWS、Kubernetes）可作为加分项
    四、行业经验
    通用要求
    初级岗位：1-3年开发经验，需参与过完整项目周期（需求分析至部署）
    高级岗位：3-5年以上经验，需主导过复杂系统设计或开源项目贡献
    垂直领域经验
    金融量化：需熟悉数字货币交易、回测框架开发及金融市场基础知识
    安防/存储：需了解硬件交互协议（如ONVIF）或分布式存储技术
